What is a dictionary in Python?

Dictionaries are powerful tools in Python. They store data in pairs, called key-value pairs. Think of a dictionary like a book where you can quickly find a word (the key) and its meaning (the value). This makes dictionaries very useful for organizing and accessing data.

Let's explore how to use dictionaries in Python!

Creating a Dictionary

To make a dictionary, we use curly braces {}. Inside, we put our key-value pairs. Each pair is separated by a comma, and the key and value are separated by a colon.

Here's an example:

my_dict = {"name": "Alice", "age": 30, "city": "New York"}

In this dictionary:

  • "name", "age", and "city" are keys
  • "Alice", 30, and "New York" are their respective values

Accessing Dictionary Values

To get a value from a dictionary, we use its key. It's like looking up a word in a real dictionary.

my_dict = {"name": "Alice", "age": 30}
print(my_dict["name"])  # This will print: Alice

Modifying Dictionaries

Dictionaries are mutable, which means we can change them after creating them.

Adding New Items

To add a new key-value pair:

my_dict = {"name": "Alice", "age": 30}
my_dict["job"] = "Engineer"
print(my_dict)  # Output: {'name': 'Alice', 'age': 30, 'job': 'Engineer'}

Updating Existing Items

To change the value of an existing key:

my_dict = {"name": "Alice", "age": 30}
my_dict["age"] = 31
print(my_dict)  # Output: {'name': 'Alice', 'age': 31}

Removing Items from Dictionaries

Sometimes we need to remove items from our dictionary. Here are three ways to do this:

  1. Using del:

    my_dict = {"name": "Alice", "age": 30}
del my_dict["age"]
print(my_dict)  # Output: {'name': 'Alice'}

  2. Using pop():

    my_dict = {"name": "Alice", "age": 30}
removed_age = my_dict.pop("age")
print(my_dict)  # Output: {'name': 'Alice'}
print(removed_age)  # Output: 30

  3. Using clear() to remove all items:

    my_dict = {"name": "Alice", "age": 30}
my_dict.clear()
print(my_dict)  # Output: {}

Iterating Over Dictionaries

We can go through each item in a dictionary using loops. This is called iterating.

Iterating Over Keys

my_dict = {"name": "Alice", "age": 30, "city": "New York"}
for key in my_dict:
    print(key)

Iterating Over Values

for value in my_dict.values():
    print(value)

Iterating Over Key-Value Pairs

for key, value in my_dict.items():
    print(f"{key}: {value}")

Checking Key Existence

Before we try to get a value, it's good to check if the key exists. This helps avoid errors.

  1. Using in:

    my_dict = {"name": "Alice", "age": 30}
print("name" in my_dict)  # Output: True
print("job" in my_dict)   # Output: False

  2. Using get():

    print(my_dict.get("name"))  # Output: Alice
print(my_dict.get("job"))   # Output: None

Nested Dictionaries

A dictionary can contain another dictionary as a value. This is called a nested dictionary.

students = {
    "Alice": {"age": 20, "grade": "A"},
    "Bob": {"age": 22, "grade": "B"}
}

print(students["Alice"]["grade"])  # Output: A

Conclusion

Dictionaries in Python are versatile and powerful. They allow us to store and access data efficiently using keys. We've learned how to create, modify, and iterate over dictionaries, as well as how to work with nested structures.

Remember these key points:

  • Dictionaries use key-value pairs
  • Keys must be unique
  • We can add, modify, and remove items easily
  • Nested dictionaries allow for complex data structures
